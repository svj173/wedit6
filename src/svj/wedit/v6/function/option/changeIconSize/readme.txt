Функционал изменения размеров иконок.

1) toolbarGeneral   - ToolBar всего фрейма
2) toolbarPanel     - ToolBar над панелями проектов, книг, текстов.
3) menu             - во всех меню - контекстных, главное меню и т.д.
4) tree             - иконки во всех деревьях
 - дерево проектов
 - дерево содержимого книг
5) system
 - button           - иконки кнопок в диалогах и пр.
