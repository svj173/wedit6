
Фрейм содержимого
- дерево проекта - разделы-книги
- вкладки с содержимом открытых книг

Структура фрейма с текстом - если открыто много глав разных книг
Слева вертикально - вкладки с названием книги
Вверху горизонтально - вкладки с названием глав (в даннйо книге).

   - конфиги
   - книги

   - списко обязательных функций - по ИД. Прописаны в FunctionCons
   
   Работа с проектом.
Меню - Проект
  - Открыть
  - Сохранить
  - Новый
  - Reopen - линки на другие проекты, которые октрываются в другом фрейме либо по закрытию текущего проекта. Список индивидуален у каждого юзера.
Проект - хранится в файле project.xml
Текущее положение - открытые книги, текущие указатели на страницах - в дин файле current_project.xml
